# Criterio Ventas

App de gestión de ventas y comisiones para Criterio Indumentaria.

## Cómo deployar en GitHub Pages

1. Creá un repositorio nuevo en github.com (puede ser privado o público)
2. Subí todos estos archivos al repositorio
3. Andá a Settings → Pages → Source: seleccioná "main" branch, carpeta "/" (root)
4. Guardá — en 1-2 minutos tu app va a estar en: `https://TU_USUARIO.github.io/NOMBRE_REPO`

## Agregar al inicio del celular

En Safari (iPhone):
- Abrí la URL de tu app
- Tocá el botón compartir (cuadrado con flecha)
- "Agregar a pantalla de inicio"

En Chrome (Android):
- Abrí la URL
- Menú (3 puntos) → "Agregar a pantalla de inicio"

## Tu API Key de Anthropic

Para que la IA lea los comprobantes necesitás una API key de Anthropic:
1. Andá a console.anthropic.com
2. Creá una cuenta (gratis para empezar)
3. Generá una API key
4. En la app, andá a Config → ingresá tu API key

La key se guarda solo en tu dispositivo (localStorage), nunca se sube a ningún servidor.
